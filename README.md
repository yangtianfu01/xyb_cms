# xyb_cms

#### 项目介绍

wagtail后台内容管理系统用于管理网站公告, 轮播图等内容

#### 环境

1. 需要安装python3.x版本
2. 需要Django2.1.3

#### 使用说明

1. 安装wagtail
    pip install wagtail
3. 安装依赖
    pip install -r requirements.txt
4. 运行项目
    python manage.py runserver
参考: [wagtail官方教程](http://docs.wagtail.io/en/v2.4/getting_started/tutorial.html).
